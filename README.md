# Expense-Tracker
Expense Tracker (Python)

Overview

The Expense Tracker is a lightweight Python-based application designed to help users record, manage, and analyze their daily expenses efficiently.
The project utilizes Pandas for structured data handling and NumPy for numerical computations, enabling users to track spending patterns and calculate total expenses with ease.

This project demonstrates the practical use of Python data analysis libraries for managing financial data in a simple and effective way.

---

Features

- Record new expenses with details such as name, amount, and category
- View all recorded expense entries
- Calculate total spending automatically
- Perform basic expense analysis using Pandas
- Efficient numerical processing using NumPy

---

Tech Stack

- Python 3
- NumPy
- Pandas

---

Project Workflow

1. The user enters expense information including expense name, amount, and category.
2. The data is stored and organized using Pandas DataFrames.
3. NumPy is used to perform numerical calculations on the expense data.
4. The program processes the stored data and displays summaries such as total expenditure.

---

Installation

Prerequisites

Make sure the following are installed on your system:

- Python 3.x
- NumPy
- Pandas

Install Dependencies

Run the following command to install the required libraries:

pip install numpy pandas

---

Usage

1. Clone the repository.
2. Navigate to the project directory.
3. Run the Python script to start tracking expenses.

python expense_tracker.py

Users can then input their expenses and view the generated summaries directly in the terminal.
