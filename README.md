# Expense-Tracker
Expense Tracker (Python)


The Expense Tracker is a lightweight Python-based application designed to help users record, manage, and analyze their daily expenses efficiently.
The project is built using Object-Oriented Programming (OOP) concepts in Python. It uses classes and objects to organize expense data and perform operations such as adding, viewing, and calculating total expenses.

This project demonstrates the practical use of Python classes, methods, and OOP design for managing financial data in a simple and structured way.

----

Features

- Record new expenses with details such as date, category, amount, and note
- View all recorded expense entries
- Calculate total spending automatically
- Display category-wise expense totals
- Organized and maintainable structure using OOP principles

----

Tech Stack

- Python 3
- Object-Oriented Programming (OOP)

----

Project Workflow

1. The user enters expense details such as date, category, amount, and note.
2. The data is stored as objects of the Expense class.
3. The ExpenseTracker class manages all expense records.
4. The program processes the stored data to display:
             - All expenses
             -Total spending
             -Category-wise spending.

----

Installation

Prerequisites

Make sure the following is installed on your system:

  - Python 3.x
  - No external libraries are required.

----

Usage

1. Clone the repository.
2. Navigate to the project directory.
3. Run the Python script to start tracking expenses.

python expense_tracker.py

Users can then input their expenses and view the summaries directly in the terminal.
